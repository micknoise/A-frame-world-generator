Procedural world examples

These scenes use algorithms ported from github.com/micknoise/world-gen (BSP dungeon + ridge dungeon), combined for a-game / A-Frame 1.3 with vendored libs. Serve the repository root over HTTP and open paths below (not file://).

Tutorial

  • Noise terrain — same stack as other examples (A-Frame 1.3, a-game, Cannon worker, a-player). Terrain mesh sits under floor so locomotion raycasts onto the surface. Optional fly-only HUD. Code: js/world/noise-terrain.js.

Indoor / dungeon generators

  • Perfect maze — recursive backtracker, BFS-farthest goal pillar, Tier A ceiling / trims / materials (see js/world/maze.js).
  • BSP + ridge carve — organic pockets grow from the BSP dungeon only where ridge noise agrees and touches existing floor.
  • BSP + ridge union — walkable if either BSP or ridge says open; then flood-filled from the BSP spawn so you stay in one connected region.
  • Ridge cavern + BSP rooms — ridge forms the cave; BSP room rectangles are forced open as anchor chambers.

Append ?seed=123 to any example URL to fix a layout. Main demo: repository root index.html (simple arena + crates).
