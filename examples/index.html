Procedural world examples

These scenes use algorithms ported from github.com/micknoise/world-gen (BSP dungeon + ridge dungeon), combined for a-game / A-Frame 1.3 with vendored libs. Serve the repository root over HTTP and open paths below (not file://).

Indoor / dungeon generators

  • Perfect maze — recursive backtracker, BFS-farthest goal pillar, Tier A ceiling / trims / materials (see js/world/maze.js).
  • BSP + ridge carve — organic pockets grow from the BSP dungeon only where ridge noise agrees and touches existing floor.
  • BSP + ridge union — walkable if either BSP or ridge says open; then flood-filled from the BSP spawn so you stay in one connected region.
  • Ridge cavern + BSP rooms — ridge forms the cave; BSP room rectangles are forced open as anchor chambers.
  • BSP noise dungeon — BSP generates the room/corridor layout; value noise displaces floor, wall, and ceiling meshes to create organic cave-like surfaces. Floors stay gently lumpy (navigable), walls bulge and recede like natural rock, ceilings sag with stalactite-like variation. All displacement amplitudes adjustable in real time. Uses js/world/noise-dungeon.js + js/world/bsp-dungeon.js.
  • BSP noise tube dungeon — same BSP graph and the same noise displacement and grain texturing as the noise dungeon, but rooms are sphere shells and corridors are open tubes (L-shaped segments). Uses js/world/noise-capsule-dungeon.js (NoiseTubeDungeon.build) + js/world/bsp-dungeon.js.
  • BSP smin cave (marching cubes) — same BSP tile layout as the noise dungeon, but the rock volume is a smooth-min union of wall columns plus ground/ceiling caps, polygonised with marching cubes, then value noise displacement along the SDF gradient (organic surface without box seams or displacement holes). Uses js/world/sdf-bsp-dungeon.js, js/world/mc-tables.js, and js/world/bsp-dungeon.js.
  • BSP smin cavern (dome + cobble + hover light) — same marching-cubes pipeline with a per-room vaulted ceiling (BSP room rects + ceiling noise; hard ceiling cut so the roof isn’t flattened by wall tops), cobblestone grain and vertex tones, noise-dungeon-style axis fBm displacement (floor / wall / ceiling sampling), and a dark scene with a dim point lamp moving above the player. Uses options on js/world/sdf-bsp-dungeon.js.

Outdoor / landscape generators

  • Noise terrain — layered value noise (fBm) with radial island falloff generates a single high-resolution smooth mesh with per-vertex terrain colouring (water → sand → grass → forest → hills → rock → snow) and a flat ocean plane. All parameters adjustable in real time. Uses js/world/noise-terrain.js.

Append ?seed=123 to any example URL to fix a layout. Main demo: repository root index.html (simple arena + crates).
