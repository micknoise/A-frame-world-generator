# GLB room (Maru)

Single-player scene: **Sketchfab “Room (Maru)”** (`room_maru.glb`, CC-BY-4.0) + **a-game** physics and locomotion, using the same vendored **`../libs/`** stack as [`arena.html`](../arena.html) (and other examples).

Open from the repo root, e.g. `http://localhost:8080/glb-room/` (or `glb-room/index.html`).

**Not included here (vs `GLB Room 2-MIMIC-2/`):** Networked-A-Frame and Firebase — those need cloud config. This page is fully offline like the procedural world.
